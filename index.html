FisiApp - Mobile Application

My group and I went out here to Bogotá looking for problems we could solve. We noticed many people with muscle injuries who couldn't find a proper way to treat them. Most of them took many medical appointments, but the recommended exercises were not easy to remember after their visit. To deal with this, we built an application where anyone can see what kind of workouts can help them depending on their injury. We also added videos showing the steps required to complete each task. The application also tracks the completed exercises using the phone's sensors to measure the user's movement. Finally, we added a map with all the medical centers near the user. The application can open a WhatsApp chat with those hospitals if needed. A notification also pops in their phone to guide them to the medical center.

Semantic Web Application

The objective of this project was to facilitate the search of papers and authors located in worldwide databases. We got the information of thousands of documents and added them into an ontology. We deployed our ontology in a graphDB server, which is accessed by our website. The site has many functionalities, such as filtering by instances, classes, or properties with SPARQL queries. Additionally, it shows graphs of the author's connections and a natural language query.

Mixed Reality Application

Virtual Reality has been an exciting topic since long ago. I built an application that could improve students' learning of all ages. Inside a virtual world, they saw a classroom with a teacher in front of them. In the classroom, a robot recited a short story, followed by a couple of questions. Later on, they did the same exercise but now with a YouTube video instead of the virtual classroom. The point of the experiment was to prove that students focus more on virtual environments.

Carpooling - Web Application

One problem many people suffer nowadays is transportation, whether it is to work or university. A possible solution to this problem is carpooling. We created an application that allows people from the same company to communicate and manage trips between them. The page connects everyone in need of partners into one place. The website was done with Meteor, React, and Firebase.

AdE - Web Application

I met with many artists in my university who had the common problem of finding others to work together. It's is a big problem because sometimes they need someone from a different area or degree. I did a website using Meteor and React where artists can post different artwork positions or apply to other's offers. The page uses Linkedin's API to create the user's profile with all the information gathered from their database.

SAVD3

Multiple sequence alignment (MSA) is a classical bioinformatics problem that compares genes or proteins to infer evolutionary history between species. My graduation project was a web application that renders various types of MSA graphs to help biologists understand the sequences they are comparing. Despite being a highly developed field, there are few modern visualization solutions, so this project was an opportunity for me to learn how genetics has evolved and transformed in recent decades. I built the website using react and D3 as the visualization engine.
